Intallation notes
=====================

Required software/libraries:

- Erlang >= 19.0
- nodejs >= 7.4.0
- npm >= 4.0.5
- elm-lang >= 1.8
- gcc, g++

Build frontend with command: `cd frontend && make`

Run backend Erlang application with command `make run`

Interface avaliable at address `http://locahost:8080`
